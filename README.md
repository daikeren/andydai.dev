# andydai.dev

My personal website built with [Astro](https://astro.build/).

## Credits

This site is based on [astro-theme-retypeset](https://github.com/radishzzz/astro-theme-retypeset) by [radishzzz](https://github.com/radishzzz).

## Development

```bash
# Install dependencies
pnpm install

# Start dev server
pnpm dev

# Build for production
pnpm build
```
